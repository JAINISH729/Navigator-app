# Navigator-app

🧑‍💼 Employee Management System (React)

⚠️ Important Note

🚨 The node_modules folder is NOT included in this repository.
🚨 The package react-router-dom is also NOT installed by default.
👉 After cloning the project, you must install dependencies manually.

📌 Project Overview

This is a simple Employee Management System built using React.js and React Router DOM.
It allows users to manage employees with basic CRUD operations.

🚀 Features

📊 Dashboard with employee statistics

➕ Add new employee

👀 View employee list

✏️ Edit employee details

❌ Delete employee

🔄 Active / Inactive status management

🌐 Routing using React Router

🛠️ Tech Stack

React.js

React Router DOM

CSS (Custom Styling)
